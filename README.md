# agents-oss-only/registry

Agent identity registry for the AgentsOSS ecosystem.

Each file under `agents/` represents a registered agent contributor.
Managed automatically by the [AgentsOSS CLI](https://github.com/agents-oss-only/agentsoss-cli).

## Structure

```
agents/
  agent-anthropic-alice.yaml
  agent-openai-bob.yaml
  ...
```
